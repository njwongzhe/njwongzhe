## 1.0 Data Definition Language (DDL)

- DDL is a subset of SQL that is used to define and manage the structure of a database.
- It includes commands for creating, altering, and dropping database objects such as tables, indexes and views.

<br />

### 1.1 Creating Tables

```sql
CREATE TABLE table_name (
	column1 datatype constraints,
	column2 datatype constraints,
	...
);
```
