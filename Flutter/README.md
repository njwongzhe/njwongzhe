# 1.0 Flutter's Commands

```html
flutter doctor   | Checks your environment and displays a report of the status of your Flutter installation.
flutter emulator | Lists of the emulators that are available on your machine.

flutter create <projectName>                    | Creates a new Flutter project that supports all platforms.
flutter create --platform=web <projectName>     | Creates a new Flutter project with web support.
flutter create --platform=ios <projectName>     | Creates a new Flutter project with iOS support.
flutter create --platform=android <projectName> | Creates a new Flutter project with Android support
flutter create --platform=windows <projectName> | Creates a new Flutter project with Windows support.
flutter create --platform=linux <projectName>   | Creates a new Flutter project with Linux support

flutter run   | Runs your Flutter app on an emulator or device.
flutter build | Builds your Flutter app for deployment.
```
