# TypeScript (.ts)

<br />

## 1.0 - What Is TypeScript & Installation

- It is a superset of JavaScript. It adds new features to JavaScript, such as static typing and interfaces.
- TypeScript code is transpiled into JavaScript code that can run in any browser or JavaScript environment.
- TypeScript Download: https://www.typescriptlang.org/download

## 2.0 - Command

```html
tsc -v | Check version of TypeScript. Use this to verify TypeScript is installed and recognized by your computer.

tsc filename.ts | Transpile TypeScript code to JavaScript. This will create a new file named filename.js in the same directory.
```
